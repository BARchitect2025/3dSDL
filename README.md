This is a project that uses SDL along with some 3d projection formulas to make a 3d game. I'm not using and graphics libraries like OpenGL or DirectX, which is why I migrated this project from Python and PyGame which was just too slow for me. Like getting 2 fps just rendering 1152 triangles. That isn't that heavy, but Python being an interpreted language just wasn't powerful enough for me. The PyGame version is in a subfolder here as well if you want to test it out. If you're new to programming, python will be a lot easier to understand than C++.
